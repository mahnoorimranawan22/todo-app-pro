# 📝 Todo App Pro

A sleek, feature-rich Task Management dashboard engineered with clean, modern ES6+ JavaScript. Designed with a strong focus on user experience, performance, and responsive UI.

🚀 **[View Live Demo Here]https://todo-app-pro-noors.netlify.app/**

---

## ✨ Core Features

* **Persistent Storage:** Automatically synchronizes all tasks, updates, and user theme preferences using browser `localStorage`.
* **Dynamic UI Tracking:** Live progress bar tracking computing `(completed / total) * 100` alongside instant statistics cards.
* **Adaptive Theme System:** Instant system-wide Dark Mode/Light Mode toggle with state memory.
* **Priority-Based Layouts:** Automated formatting for High, Medium, and Low priority sorting badges.
* **Real-Time Substring Search:** Interactive `keyup` searching mechanism prioritizing zero latency.
* **Custom Toast Architecture:** Non-blocking reusable toast notifications confirming user actions.

---

## 🛠️ Tech Stack Used

* **HTML5:** Semantic architecture with dedicated structural layouts.
* **CSS3:** Flexbox/Grid systems, responsive breakpoints, custom theme variables.
* **JavaScript (ES6+):** Dynamic DOM Manipulation, state encapsulation, window-scoped handlers, error-resilient JSON processing.

## 📸 How to Run the Project Locally

1. Clone or download this repository.
2. Open the directory containing the files.
3. Simply launch `index.html` in any modern web browser, or use the VS Code **Live Server** extension.
